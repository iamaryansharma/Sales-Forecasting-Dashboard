# 📊 AI-Based Sales Forecasting and Business Analytics Dashboard

## 📌 Overview

This project is an AI-powered Sales Forecasting and Business Analytics Dashboard developed using Streamlit and Machine Learning. It allows users to upload sales datasets, analyze business performance, visualize trends, and forecast future sales.

---

## 🚀 Features

- Upload CSV or Excel Dataset
- Automatic Column Detection
- Dataset Preview
- Business Insights
- KPI Dashboard
- Revenue Trend Analysis
- Monthly Sales Analysis
- Category-wise Analysis
- Product-wise Analysis
- Location-wise Analysis
- Payment Mode Analysis
- Correlation Heatmap
- 90-Day Sales Forecast
- Download Forecast CSV
- Download Filtered Dataset

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn
- OpenPyXL
